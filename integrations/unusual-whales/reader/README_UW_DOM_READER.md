# UW DOM Reader

Status: not implemented in this phase.

This directory is reserved for the future local DOM Reader that will:

- open logged-in Unusual Whales pages in the user's local browser context
- read visible DOM fields only
- produce summarized UW snapshots
- avoid storing cookies, raw member HTML, or account credentials

Not implemented yet:

- page automation
- selector extraction
- screenshot pipeline
- raw DOM persistence
- reader runtime

This PR only prepares the directory and documentation boundary.
